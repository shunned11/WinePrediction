# Introduction
This project was created as and assignment for an internship. It predicts the variety of wine based on text reviews

# Dependecies
* Python 3.7.7
* Pandas 1.0.3
* Keras 1.1.0
* Scikit-learn 0.22.1
* Tensorflow 2.1.0

# Dataset
*The train-dev set consisted of 82657 examples

*The test set consisted of 20665 examples
